# Developer Portfolio Starter (Simplified)

![Portfolio Screenshot](https://assets.stackbit.com/docs/personal-nextjs-starter-thumb.png)

This is a portfolio website using Next.js and Tailwind CSS. It includes features like annotations for components visible in the Netlify Visual Editor.

**Live Demo:** [auto-annotated-portfolio.netlify.app](https://auto-annotated-portfolio.netlify.app)

## Quick Start

1. Clone the repo and install dependencies:
   ```bash
   git clone <repo-url>
   cd auto-annotated-portfolio
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Install and launch Netlify Visual Editor:
   ```bash
   npm install -g @stackbit/cli
   stackbit dev
   ```
4. Open the generated Visual Editor URL and register/sign in.
